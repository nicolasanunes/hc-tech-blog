## Descrição

Projeto backend de desenvolvimento de uma API REST para servir o frontend de um blog de compartilhamento de conhecimento. Projeto motivado pelo teste técnico da empresa Grão Direto em jan/2025.

## Instalar dependências do projeto

```bash
$ npm install
```

## Compilar e rodar o projeto

```bash
# desenvolvimento
$ npm run start

# modo assistido
$ npm run start:dev

# produção
$ npm run start:prod
```

## Executar testes

O presente projeto não conta com testes implementados, e2e ou initários. Mas os scripts para executá-los encontram-se registrados:

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
