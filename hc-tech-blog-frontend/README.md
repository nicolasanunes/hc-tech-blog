# Descrição

Desenvolvimento da parte frontend do teste técnico da empresa Grão Direo em jan/2026.

## Instalar dependências do projeto

```sh
npm install
```

### Compilar e rodar o projeto

```sh
npm run dev
```

### Validar tipos, compilar e gerar /dist para produção

```sh
npm run build
```

### Lint do projeto com [ESLint](https://eslint.org/)

```sh
npm run lint
```
